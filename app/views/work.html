COOKESLY
SU WATS 2000 Final Project

A fully functional web application to search, browse, and save 1000's of cooking recipes.

Live App Source Code

Desktop Views

Syponsis

The goal of the course project was to build a polished data driven application utilizing a web API and AngularJS framework. The application took a total of 4 weeks from concept to build. I had originally spec for Eventbrite API integration so that a user can host an Eventbrite event by selecting a recipe within the application. I scrapped that idea 2 weeks in the build and opted for the save/delete recipe function due to time constraints. The name COOKESLY for the application meant cook easily which I thought suited the purpose of the application.
